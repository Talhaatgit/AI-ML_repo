## Concepts Covered

1. Displaying output using the `print()` function  
2. Accepting user input using the `input()` function  
3. Understanding variables and dynamic typing  
4. Introduction to basic data types (`int`, `float`, `str`, `bool`)  
5. Type conversion using built-in functions (`int()`, `float()`, `str()`)  
6. Introduction to arithmetic, comparison, logical, and assignment operators

## Learning Objectives

Upon completing this section, learners should be able to:

- Use the `print()` and `input()` functions appropriately  
- Declare, assign and modify variables  
- Identify and use basic Python data types correctly  
- Convert data from one type to another where necessary  
- Apply arithmetic, comparison, logical and assignment operators  
- Interpret basic program output and understand the underlying logic
